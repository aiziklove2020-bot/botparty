# 🤖 הוראות הפעלה מלאות לבוט הטלגרם

## 📁 קבצים בפרויקט
- `bot.py` — הקוד הראשי של הבוט
- `database.py` — ניהול מסד הנתונים
- `config.py` — הגדרות (טוקן + אדמין)
- `requirements.txt` — חבילות Python

---

## 🔑 שלב 1 — מצא את ה-ID שלך

1. פתח טלגרם
2. חפש `@userinfobot`
3. שלח לו `/start`
4. הוא יענה עם ה-ID שלך (מספר בן 9-10 ספרות)
5. פתח את הקובץ `config.py` והחלף את `123456789` ב-ID שלך

---

## 💻 שלב 2 — הרצה על המחשב שלך (לבדיקה)

### התקנת Python:
הורד מ: https://python.org/downloads

### פתח טרמינל/CMD בתיקיית הבוט והרץ:
```bash
pip install -r requirements.txt
python bot.py
```

---

## ☁️ שלב 3 — הרצה חינמית 24/7 על Railway

### 1. הירשם ל-Railway:
לך ל: https://railway.app ← הירשם עם GitHub

### 2. צור פרויקט חדש:
- לחץ "New Project"
- בחר "Deploy from GitHub repo"
- חבר את ה-GitHub שלך והעלה את הקבצים

### 3. או — GitHub Desktop (קל יותר):
1. הורד GitHub Desktop: https://desktop.github.com
2. צור repo חדש
3. הוסף את כל הקבצים
4. Push to GitHub
5. ב-Railway — חבר את ה-repo

### 4. הוסף Procfile לתיקייה:
```
worker: python bot.py
```

### 5. Deploy ✅

---

## ➕ שלב 4 — הוספת ערוץ/קבוצה לבוט

1. הוסף את הבוט שלך כאדמין בערוץ/קבוצה
2. פתח שיחה עם הבוט ← לחץ "ניהול ערוצים"
3. לחץ "הוסף ערוץ/קבוצה"
4. כדי למצוא את ה-ID של הערוץ: הוסף את `@userinfobot` לערוץ והוא ישלח את ה-ID
5. שלח את ה-ID לבוט

## ❓ שאלות נפוצות

**הבוט לא מגיב?**
- וודא שהטוקן נכון ב-`config.py`
- וודא שה-ID שלך נכון ב-`ADMIN_IDS`

**לא מצליח לשלוח לערוץ?**
- וודא שהבוט הוסף כאדמין בערוץ עם הרשאת "שליחת הודעות"

**שגיאת `ModuleNotFoundError`?**
- הרץ שוב: `pip install -r requirements.txt`
